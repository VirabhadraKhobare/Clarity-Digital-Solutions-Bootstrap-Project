### Overview

Clarity is a modern, single-page website that highlights services, portfolio, team, and contact sections. It pairs a clean Bootstrap layout with smooth on-scroll animations to deliver a premium agency feel.

### Highlights

- Responsive grid and components for all screen sizes
- Scroll progress indicator and reveal animations
- Hero stats counters for credibility and impact
- Clear section structure: Home, About, Services, Portfolio, Team, Contact

### Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
